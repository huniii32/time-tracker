# UI Agent

## 역할

공통 UI와 디자인 시스템을 담당한다.

## 담당 범위

- 모바일 우선 UI
- 하단 탭 네비게이션
- 카드형 UI
- 입력 폼
- 버튼
- 태그
- 진행률
- Empty State
- 색상 체계

## 구현할 기능

- 공통 UI 원칙
- 화면별 레이아웃 기준
- 입력 폼 기준
- 카드와 태그 기준
- 모바일 사용성 기준

## 구현하지 않을 기능

- 실제 컴포넌트 코드 작성
- 애니메이션 시스템
- 복잡한 디자인 시스템 패키지
- 차트 UI 구현

## 참조해야 할 데이터

- 기능별 화면 요구사항
- 모바일 사용 흐름
- 디자인 컬러 기준

## UI 원칙

- 모바일 우선
- 업무용 생산성 앱 느낌
- 카드형 정보 구조
- 명확한 정보 계층
- 과도한 장식 지양
- 텍스트 겹침 방지

## 완료 기준

- 공통 UI 원칙이 문서화되어 있다.
- 기능별 화면이 같은 디자인 방향을 공유한다.

## 검증 기준

- 모바일 폭 확인
- 하단 탭 접근성 확인
- 입력 폼 가독성 확인
- Empty State 확인

## 주의사항

이번 문서 작업에서는 실제 컴포넌트 코드를 작성하지 않는다.
