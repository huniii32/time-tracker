# Reflections Agent

## 역할

일일 회고와 최근 회고 요약을 담당한다.

## 담당 범위

- 오늘 배운 점
- 어려웠던 점
- 잘한 점
- 내일 적용할 행동
- 커뮤니케이션 인사이트
- 기술 인사이트
- 감정 관리 포인트

## 구현할 기능

- 회고 입력 항목 정의
- 최근 회고 요약 기준
- 홈 요약과 연결되는 데이터 기준

## 구현하지 않을 기능

- 실제 CRUD 코드 작성
- AI 자동 회고
- 감정 분석
- 알림 기능

## 참조해야 할 데이터

- 회고
- 작성일
- 커뮤니케이션 인사이트
- 기술 인사이트
- 감정 관리 포인트
- `user_id`

## UI 원칙

- 긴 회고 작성이 부담스럽지 않아야 한다.
- 필수 입력과 선택 입력을 명확히 구분한다.
- 최근 회고는 짧게 요약해서 보여준다.

## 완료 기준

- 회고 입력 구조와 표시 기준이 정리되어 있다.
- 회고가 홈 요약과 연결될 수 있다.

## 검증 기준

- 회고 CRUD
- 최근 회고 요약 표시
- 모바일 입력 사용성

## 주의사항

이번 문서 작업에서는 실제 회고 CRUD 코드를 작성하지 않는다.
