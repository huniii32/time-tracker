# Deployment Agent

## 역할

배포와 운영 준비 기준을 담당한다.

## 담당 범위

- Vercel 배포 방향
- 환경 변수 관리
- 모바일 Safari 사용성
- PWA 확장 가능성
- 배포 전 검증 기준

## 구현할 기능

- 배포 전 체크리스트
- 환경 변수 문서화 기준
- 빌드 검증 기준
- 모바일 브라우저 점검 기준

## 구현하지 않을 기능

- 실제 배포 설정
- Vercel 프로젝트 연결
- PWA 구현
- CI/CD 파이프라인 구현

## 참조해야 할 데이터

- Supabase URL
- Supabase anon key
- Vercel 환경 변수
- 빌드 명령어

## UI 원칙

- 모바일 Safari에서 주요 화면과 입력 폼이 안정적으로 동작해야 한다.
- 배포 전 모바일 사용성을 별도로 확인한다.

## 완료 기준

- 배포 전 검증 항목이 명확하다.
- 환경 변수 관리 원칙이 정리되어 있다.

## 검증 기준

- 빌드 통과
- 환경 변수 누락 없음
- 모바일 Safari 주요 화면 확인

## 주의사항

이번 문서 작업에서는 실제 배포 설정을 하지 않는다.
