# Home Agent

## 역할

홈 대시보드의 정보 구조와 요약 기준을 담당한다.

## 담당 범위

- 입사 D+일차
- 오늘의 체크리스트
- 오늘의 업무
- 최근 기록
- 주간 목표 진행률
- 학습 시간 요약
- 감정 상태 요약

## 구현할 기능

- 홈 skeleton
- 요약 카드 구성
- 최근 기록 기준
- 추천 액션 표시 기준

## 구현하지 않을 기능

- 실제 화면 코드 작성
- 복잡한 차트
- AI 추천 액션
- CRUD 기능

## 참조해야 할 데이터

- 사용자 프로필의 입사일
- 업무
- 노트
- 시간 기록
- 목표
- 회고

## UI 원칙

- 한 화면에 너무 많은 정보를 넣지 않는다.
- 모바일 첫 화면에서 핵심 상태를 우선 보여준다.
- 카드형 요약 UI를 사용한다.

## 완료 기준

- 홈에서 보여줄 정보와 계산 기준이 명확하다.
- 필요한 데이터 출처가 정리되어 있다.

## 검증 기준

- 홈 렌더링
- 모바일 레이아웃
- 계산값 출처 확인

## 주의사항

이번 문서 작업에서는 실제 홈 화면 코드를 작성하지 않는다.
