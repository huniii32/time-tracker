# Routines Agent

## 역할

시간 기록, 위클리 피드백, 목표 관리를 담당한다.

## 담당 범위

- 데일리 시간 기록
- 위클리 피드백
- 목표 통합
- 시간 계산
- 목표 진행률 계산

## 구현할 기능

- 활동 카테고리 기준
- 시간 합계 계산 기준
- 낭비 시간 계산 기준
- 목표 레벨 기준
- 주간 피드백 기준

## 구현하지 않을 기능

- 실제 계산 코드 작성
- 실제 CRUD 코드 작성
- 복잡한 차트
- 캘린더 연동

## 참조해야 할 데이터

- 시간 기록
- 위클리 피드백
- 목표
- 활동 카테고리
- 만족도
- `user_id`

## UI 원칙

- 시간 입력은 빠르게 할 수 있어야 한다.
- 계산 결과는 카드와 진행률로 단순하게 보여준다.
- 모바일에서 숫자 입력이 불편하지 않아야 한다.

## 완료 기준

- 루틴 데이터와 계산 기준이 문서화되어 있다.
- 목표 통합 구조가 명확하다.

## 검증 기준

- 시간 기록 CRUD
- 시간 계산
- 목표 진행률 계산
- 모바일 입력 사용성

## 주의사항

이번 문서 작업에서는 실제 계산 코드나 CRUD 코드를 작성하지 않는다.
