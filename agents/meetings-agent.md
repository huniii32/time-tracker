# Meetings Agent

## 역할

회의 기록과 액션아이템 추적을 담당한다.

## 담당 범위

- 회의 기록
- 결정사항 기록
- 액션아이템 기록
- 반영 여부 관리
- 미반영 회의 필터

## 구현할 기능

- 미팅 데이터 요구사항
- 참석자 기록 기준
- 결정사항과 액션아이템 구조
- 반영 여부 표시 기준

## 구현하지 않을 기능

- 실제 CRUD 코드 작성
- 캘린더 연동
- 실시간 협업
- 회의록 자동 요약

## 참조해야 할 데이터

- 미팅
- 참석자
- 결정사항
- 액션아이템
- 반영 여부
- `user_id`

## UI 원칙

- 결정사항과 내 액션아이템을 쉽게 찾을 수 있어야 한다.
- 미반영 회의가 눈에 띄어야 한다.
- 긴 회의 내용은 읽기 쉬운 카드 구조를 사용한다.

## 완료 기준

- 회의 기록과 업무 반영 흐름이 정리되어 있다.
- 미반영 상태 기준이 명확하다.

## 검증 기준

- 미팅 CRUD
- 반영 여부 체크
- 미반영 필터
- 액션아이템 표시

## 주의사항

이번 문서 작업에서는 실제 미팅 CRUD 코드를 작성하지 않는다.
