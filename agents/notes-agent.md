# Notes Agent

## 역할

회사 적응에 필요한 개인 노트 기능을 담당한다.

## 담당 범위

- 회사노트
- 상사노트
- 회사 단어 사전
- 동료노트
- 감정노트
- 학습노트
- 노트 CRUD
- 검색
- 태그

## 구현할 기능

- 노트 타입별 입력 항목 정의
- 노트 목록 기준
- 검색 및 태그 기준
- 날짜 정렬 기준

## 구현하지 않을 기능

- 실제 CRUD 코드 작성
- AI 요약
- 파일 첨부
- 공유 기능

## 참조해야 할 데이터

- 노트
- 노트 타입
- 태그
- `user_id`
- 생성일과 수정일

## UI 원칙

- 노트 타입 선택이 명확해야 한다.
- 긴 텍스트 입력이 모바일에서 불편하지 않아야 한다.
- 태그는 작고 읽기 쉽게 표시한다.

## 완료 기준

- 노트 타입별 요구사항이 분리되어 있다.
- 본인 데이터만 접근하는 원칙이 유지된다.

## 검증 기준

- 노트 생성, 조회, 수정, 삭제
- 검색
- 태그 표시
- 사용자 데이터 격리

## 주의사항

이번 문서 작업에서는 실제 노트 CRUD 코드를 작성하지 않는다.
